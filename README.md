# Bigger Cellars Rebuilt

A Vintage Story mod that allows larger valid rooms to count as cellars.

## Overview

This repository contains the source code and project files for Bigger Cellars Rebuilt.

If you are visiting this page as a player, check the releases page for downloadable builds when they are published.

## Features

- Expands the room-size limits used for cellar validation
- Rebuilds cellar-size logic with Harmony patches
- Keeps larger storage spaces usable as proper cellars


Issues and pull requests are welcome. For larger changes, opening an issue first is helpful.
